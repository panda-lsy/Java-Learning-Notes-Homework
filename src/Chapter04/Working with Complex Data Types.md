# 数组

数组是存储在相邻内存位置的单一数据类型元素集合，可通过指定数组的名称&下标访问

指定数组中元素的位置，成为元素的索引

| 内存地址 | 25001 | 25002 |
| -------- | ----- | ----- |

## 分类

### 一维数组

- 声明一个数组
- 为数组赋值

```java
arraytype arrayname[] = new arraytype[size]
int [] a = new int[3];
String [] jumbledWords = new String[3];
```

| a[0] | a[1] | a[2] |
| ---- | ---- | ---- |
| 101  | 102  | 103  |

### 二维数组

```java
int a[][] = new int[2][2];
int a[][] = {{1,2},{3,4}}
```
